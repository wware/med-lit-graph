About This API

The Medical Literature Graph API is a provenance-first knowledge graph built on PubMed/PMC literature. Every relationship in the graph is traceable to specific evidence from peer-reviewed research papers.

This demo interface allows you to interactively test the API by:

  • Selecting from pre-built query examples
  • Editing and customizing queries
  • Viewing formatted JSON responses
  • Exploring multi-hop graph queries
Available Endpoints:
  • POST /api/v1/query - Main query endpoint
  • GET /docs - Interactive API documentation
  • GET /api/v1/stats - Graph statistics
