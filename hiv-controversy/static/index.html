About This Graph Database

This is a provenance-first knowledge graph built from PMC literature about the HIV/AIDS controversy. The graph is stored in PostgreSQL with the Apache AGE graph extension, allowing powerful Cypher queries to explore relationships between papers, claims, entities, and evidence.

This interface allows you to:

  • Select from pre-built Cypher query examples
  • Edit and customize queries to explore the data
  • View formatted JSON responses with syntax highlighting
  • Trace claims back to their source papers and evidence
Graph Schema:
  • Nodes: Paper, Entity, Paragraph, Claim, Evidence
  • Edges: CONTAINS, MAKES_CLAIM, CONTAINS_CLAIM, SUPPORTS, HAS_SUBJECT, HAS_OBJECT
  • Query Language: Cypher (via Apache AGE)
